# Olympus UI

为 Web3 应用注入神话美学的 React UI 组件库。


## ✨ 特性

*   **🏛️ 神话设计系统**：汲取神话灵感，提供独特的视觉语言与设计令牌（Design Tokens）。
*   **🔗 为 Web3 而生**：内置钱包地址格式化、交易状态、网络切换等场景化组件与工具。
*   **🎨 深度可定制**：通过 CSS 自定义属性（CSS Variables）轻松实现全局主题切换与样式覆盖。
*   **⚡ 开发体验优先**：使用 TypeScript 构建，提供完整的类型提示。基于 Vite 构建，支持 Tree Shaking。
*   **🧩 CSS Modules 封装**：样式具有天然的隔离性，避免在复杂应用中发生冲突。


## 📦 安装

```bash
# 使用 npm
npm install @olympus/ui

# 使用 yarn
yarn add @olympus/ui

# 使用 pnpm
pnpm add @olympus/ui
